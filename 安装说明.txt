淘宝购物车导出助手  v1.0.0（探针版）  —  安装说明
====================================================

【这是什么】
一个独立 Chrome 扩展，把淘宝购物车勾选的商品导出为 Excel(.xlsx)，6 列：
    名称 | 规格 | 链接 | 数量 | 紧急程度 | 图片
点「导出购物车」后弹窗二选一：
    🖼️ 导出嵌入 —— 真·嵌单元格（Excel 365 / 新版 WPS）
    📦 导出浮动 —— 浮动图（所有 WPS / 所有 Excel，兼容性最好）

【本包内容】
    taobao-cart-export/    ← 扩展本体（加载这个文件夹）
    README.md / 安装说明.txt

【系统要求】
- Chrome / Edge / Brave（Chromium 111+，Mac / Windows / Linux 通用）
- 打开 xlsx 用 Excel 365 / 新版 WPS（嵌入模式）；旧版用浮动模式


【安装步骤】
1. 把本压缩包解压到一个**固定位置**（别放下载/桌面等临时目录），例如：
     Windows: D:\Extensions\taobao-cart-export
     Mac:     ~/Documents/taobao-cart-export
   解压后这个文件夹**不能删、不能移动**。

2. 打开 Chrome（或 Edge / Brave），地址栏输入：  chrome://extensions

3. 右上角打开「开发者模式」开关。

4. 点左上角「加载已解压的扩展程序」，选中里面的  taobao-cart-export  文件夹。

5. 打开 https://cart.taobao.com/  → 勾选商品 → 点页面右下角橙色「📥 导出购物车」按钮
   → 弹窗选「导出嵌入」或「导出浮动」→ 下载 .xlsx


【⚠️ 探针版必做：发一次日志】
本版是探针版——淘宝购物车的数据结构需按你的真实页面精准适配一次。
第一次用：
  1) 在购物车页按 F12 → Console 标签
  2) 上下滚动一下购物车（让数据加载）
  3) 点右下角「导出购物车」
  4) 把控制台里 [购物车导出] 开头的日志（尤其红色错误那段）复制/截图发给作者
  5) 作者据此出正式版（届时无需再发日志）


【常见问题】
Q: 点导出弹"没抓到勾选的商品"？
A: 探针版正常现象。按上面「探针版必做」把 F12 日志发作者即可。

Q: 图片列是空的或 #VALUE!？
A: 用了「嵌入」但打开的软件不支持 Rich Data（旧版 WPS/旧版 Excel）。换「导出浮动」重导一次。

Q: 每次启动浏览器弹「已停用开发者模式扩展」黄条？
A: 点「保留」即可，Chrome 对所有非商店扩展的例行警告，不影响使用。
